<h3>7 皇帝发飙</h3>
&nbsp;&nbsp;&nbsp;&nbsp;他这一骂，把底下这些阁臣各自的小算盘都骂得不敢打了。平台的气氛有点凝重，就连呼吸都不敢稍微大声。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;胡广要的就是这个效果，看看火候差不多了，便缓和了点语气说道：“不管祖大寿是何种理由，此皆为大罪！然朕正在用人之际，他祖大寿要能认罪，悔过自新，并立下大功，朕可以不再追究。”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;韩爌等人一听，心中不由得无语。刚才还以为皇上不顾大局，真要任性追究祖大寿，要逼反他，没想这口气一转，其实也没有追究的意思，只是要有个台阶可下而已。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;他们这些人都是玩弄文字的高手，也很快明白过来，崇祯皇帝先怒后开恩之举，其实还是很厉害的。这传出去后，恐怕没有第二个武夫敢效仿。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;他们正想着，就听到崇祯皇帝的语气又转严厉了一点道：“如若他不知道好歹，枉顾君恩，不思精忠报国，朕就不信关宁军中，个个都会从贼，朕有的是治他的手段！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;关宁军的后勤就掌握在朝廷手中，只要断了供应，要么兵溃，要么造反，要么投靠建虏，除此之外，没有其他路可走。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;这三条路，或者可以隐隐以此要挟下朝廷，却没人敢在走投无路之前就走这样的路。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;究其原因，是已有先例。就明朝来说，英宗时期皇帝被抓，蒙古鞑子的大军携大胜之威兵临京师城下，比起眼下还要危急，可最后还不是分崩离析，而大明却延续国运至今。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;这样的道理，人人都懂。可真要这么做，却是要有过人的胆识，能承担万一的后果。这些文臣中，没人有这个胆识。就算以前的崇祯，也没这个魄力，但胡广有！（胡广的独白：我也是被逼的，完不成任务要被抹杀啊！)
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“叮，成就值+1，来自首辅韩爌！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“叮，成就值+1，来自次辅李标！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“叮，成就值+1，来自阁臣钱龙锡！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“叮，成就值+1，来自阁臣成基命！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“叮，成就值+1，来自阁臣周延儒！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;聊天群左下角位置：成就值+31。胡广一见，有点懵了，连忙在脑海中问系统道：“系统，怎么才给一点成就值，你出bug了？”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“系统提示：同一人从第二次开始的贡献值一律只能奖励1点。”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“系统，你这不讲理啊！这样一来，优秀的人岂不是很吃亏？”胡广不甘心地争辩道。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“规则如此，不能更改！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“你妹！”胡广骂了一句，知道拿这个死脑筋的程序没办法，从脑海中退出来，注意力集中到眼前。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;对于关宁军的事儿，基调已经定好了。就处罚为首的祖大寿，但他要是能及时认罪，并立功赎罪的话，可以宽恕。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;胡广整理了下思路，又严肃而痛心地说道：“建虏肆虐京畿之地，所过之处，生灵涂炭，朕心甚忧！卿等可有良策，给建虏一个教训，及早驱出京畿之地，甚至把建虏消灭在关内？”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;一听这话，这些阁臣顿时愣住了。之前他们中还有人在担心京师不保的，谁知皇上竟然要求还要更进一步，要给建虏一个教训，甚至还想消灭建虏在关内，这岂不是白日做梦么，疯了才会这么想！
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;一想到这，他们马上想起之前宫里传出来的话。顿时都吃了一惊，该不会是皇上真失心疯了吧？
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;不管是作为首辅的韩爌，还是想一心夺回首辅位置的次辅李标，震惊于心中的想法，一个个不敢出声。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“怎么，一个个都哑巴了？”胡广的脸沉了下来，冷声喝问道。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;这一下，众人之首的韩爌不得不开口了：“陛下，建虏势大，不可轻举妄动啊！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“臣亦以为然，陛下连日操劳，龙体已是欠安，当好生静养为好！”钱龙锡也连忙跟进劝道。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;周延儒也怕了，万一皇上疯了干出疯狂的事来，后果不堪设想，便也赶紧附议道：“陛下安心静养龙体，等关宁军归，勤王大军云集，那时再商讨此事，才是万全之策！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;李标等人虽然没出声，却都抬头看着崇祯皇帝，眼神中也是赞同之意。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;胡广听了，有点不解，打量了下他们，忽然回过神来，该不会是他们已经知道早上在宫里发生的事情了吧？
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;你妹的，一个个都在朕身边安排了眼线，真当朕是好欺负的？胡广怒了，不过他没马上发飙，只是喝问道：“敢情朕说话没人听了是吧？要不要朕再说一次？”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;皇帝坚持，作为臣子的就没折了，一个个担心的同时，都把目光集中到了首辅身上。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;韩爌无法，只好奏道：“陛下，此事当召兵部尚书和总督京师勤王兵马的满总兵问之为上策！”
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;“传！”胡广言简意赅，语气很不满意。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;之后，他看着底下跪着的这些阁臣，也没打算让他们起来。太让人失望了，没一个能让自己满意，就跪着好好反省吧！
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;就在等候的时候，有通政司送来奏章。胡广随手翻了几本。因为融合了原崇祯皇帝的灵魂，因此他对古代竖版繁体无标点的奏章看着也没什么不习惯，看来看去，不是这个说缺饷，就是那个说缺粮，又或者报京师难民冻死多少人……反正没一件是好事。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;忽然，胡广总算看到一本奏章，脸上露出了笑容，特意从那一堆奏章中拿出来放到一边。这让那些偷偷观察的阁臣一个个很纳闷，不知道那本奏章是谁的，又说了什么，竟然会让皇上露出笑容？
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;就这样，一直等到了总兵满桂和兵部尚书申用懋入内。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;兵部尚书原为王洽，因为兵事刚被下狱，兵部左侍郎申用懋接任兵部尚书。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;此时申用懋虽有边才之名，却已是七十岁的年龄，面对如今的危局，心有余而力不足，闻皇帝召对，心中很是惶恐。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;倒是满桂有点与众不同，有点符合胡广印象中武将的形象。个子大概就一米七五，不算高。长得很壮实，满脸胡子，大步走路，赳赳武夫的样子。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;两人进来后，看到一众阁臣都跪在那里，都有点吃惊，连忙给崇祯皇帝见礼。
<br/>
<br/>&nbsp;&nbsp;&nbsp;&nbsp;于是，崇祯皇帝又把前面说的话重新说了一边后道：“卿等可有良策？”     <br />
